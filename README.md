# Pollution Management System

## Overview
The Pollution Management System is a web-based application developed to manage and monitor pollution-related records. It provides an easy way to store, update, and view pollution data.

## Technologies Used
- Node.js
- Express.js
- JavaScript
- HTML
- CSS
- MySQL

## Features
- Pollution data management
- User-friendly interface
- Database connectivity
- Record storage and retrieval
- Web-based access

## Project Structure
- server.js : Main server file
- db.js : Database connection
- routes/ : Application routes
- views/ : HTML pages
- public/ : CSS and static files

## Installation
1. Clone the repository
2. Install dependencies:
   npm install
3. Configure the database
4. Start the server:
   node server.js
